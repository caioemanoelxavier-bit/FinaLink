# 💰 FinaLink — Controle Financeiro

> Aplicação web para registrar receitas e despesas e manter o controle das finanças pessoais.

## 📋 Descrição

FinaLink é um app de controle financeiro pessoal que permite registrar entradas e saídas de dinheiro, visualizar o saldo atual e acompanhar para onde o dinheiro está indo — tudo de forma simples e visual.

## ⚙️ O que o app faz

- Registra receitas e despesas
- Calcula e exibe o saldo atual automaticamente
- Categoriza os lançamentos financeiros
- Permite visualizar o histórico de transações
